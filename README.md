# X-Mass-Dislike-Script
Updated Version of a Free X (Twitter)) Mass dislike Script
How to use it:
Navigate to your Likes tab on your X profile.
Open the Developer Tools (F12 or Right Click -> Inspect).
Go to the Console tab.
Paste the code above and hit Enter.
Keep the tab open and active. If you switch tabs, modern browsers will throttle the JavaScript, and the script will pause or fail.
